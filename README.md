Explicacion: Este programa ha sido creado en c++, Este esta hecho para permitir que el usuario ingrese una cantidad de numeros enteros.
hay es donde el programa una vez ingresado los numeros muestre cuantos son positivos, Negativos, Pares e impares.

El programa funciona primero pidiendo al usuario que escriba la cantidad de numeros de desea ingresar, Esto sirve para saber cuantas veces se repetira el proceso.
Despues  el programa solicitara los numeros uno por uno, Cuando todo este hecho el sistema evaluara que es el numero.
Durante el proceso el programa acumulara la cantidad de numeros en diferentes contatores para que se pueda saber de maneras mas facil cuantos de cada tipo

las estructuras que se usaron en este programa fueron if y else para las estructuras condicionales, For para los ciclos y % pra identificar cuales era impar y par
